TEAM TASK MANAGER (FULL-STACK)

Project Overview:
Team Task Manager is a full-stack web application developed to help teams manage projects, assign tasks, and track progress efficiently using role-based access control.

Features:
- User Authentication (Signup/Login)
- Role-Based Access (Admin/Member)
- Project Creation and Management
- Task Creation and Assignment
- Task Status Tracking
- Dashboard with Task Summary and Overdue Tasks
- Secure REST APIs
- Form Validation and Error Handling

Roles:

1. Admin
- Create and manage projects
- Add team members
- Assign tasks
- Monitor task progress

2. Member
- View assigned tasks
- Update task status
- Track personal progress
